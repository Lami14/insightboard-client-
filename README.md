# InsightBoard Client

Frontend client for InsightBoard – a SaaS-style Admin Dashboard built with React, protected routes, role-based UI, and analytics charts.

## Tech Stack
- React
- React Router
- Axios
- Recharts
- Tailwind CSS
- Context API

## Features
- Login / Register
- Authentication & JWT-based session
- Protected routes & role-based access
- Dashboard analytics with charts
- User management (admin)
- Dark / Light mode toggle
- 404 page handling
